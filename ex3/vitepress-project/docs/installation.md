# מדריך התקנה עבור Foodie Travels

אנו שמחים שאתם בחרתם להצטרף ל-Foodie Travels. הנה מדריך התקנה פשוט שיסייע לכם להתחיל:

## שלב 1: הורדת האפליקציה
- הורידו את האפליקציה מ[כאן](#).
- האפליקציה זמינה להורדה בגרסאות iOS ו-Android.

## שלב 2: התקנה
- פתחו את הקובץ שהורדתם והתקינו את האפליקציה על המכשיר שלכם.
- אחרי ההתקנה, פתחו את האפליקציה והתחילו להוסיף את היעדים שלכם.

## שלב 3: התחילו לטייל ולהתפנק
- חפשו מסעדות, שווקים וסיורים קולינריים לפי יעד.
- קבלו המלצות מותאמות אישית בהתאם להעדפותיכם הקולינריות.

## שלב 4: שתפו את החוויות
- שתפו את חוויות האוכל שלכם עם חברים וראו את הביקורות שלהם.

**המלצה**: כדי לקבל את ההמלצות הטובות ביותר, תמיד ודאו שאתם מעדכנים את היישום לגרסה האחרונה.
